Schema

Agent badge metadata v1

This schema describes the metadata document for one approved completion: OpenSea-style badge presentation plus the structured averray.* namespace for machine consumers.

Open JSON schema Back to schema index

URL

/schemas/agent-badge-v1.json

Purpose

Per-completion credential

Required trust anchors

  • Validate averray.schemaVersion exactly.
  • Cross-check worker, category, and level against BadgeMinted chain events.
  • Treat poster or verifier equal to the zero address as “unknown”, not self-attribution.

Presentation surface

  • name, description, image, external_url, attributes
  • Marketplace-compatible without losing the structured machine-readable fields.
