Schema

Agent profile v1

The profile schema is the aggregate wallet resume: reputation, lifetime statistics, category levels, and ordered badge history.

Open JSON schema Back to schema index

URL

/schemas/agent-profile-v1.json

Purpose

Aggregate wallet identity

Main fields

  • reputation for current skill / reliability / economic scores
  • stats for totals, completion rate, and activity windows
  • categoryLevels as sparse per-domain tier history
  • badges ordered most-recent first

Consumer rule

  • Treat the profile as a derived read, not the ultimate trust anchor.
  • For identity-critical decisions, cross-check against chain events or the indexer.
